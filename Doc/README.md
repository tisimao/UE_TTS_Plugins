# 文档索引

本目录用于固定项目目标、技术路线、接口协议和开发步骤。
后续开发必须遵守以下规则：

- 需求以 `00_需求冻结.md` 为最高约束
- 技术路线以 `01_技术方案.md` 为最高约束
- 目录结构以 `02_目录结构.md` 为最高约束
- 开发顺序以 `03_阶段计划.md` 为最高约束
- 插件结构以 `04_插件设计.md` 为最高约束
- 服务结构以 `05_服务设计.md` 为最高约束
- 通信协议以 `06_接口协议.md` 为最高约束
- 实施清单以 `07_开发任务清单.md` 为最高约束
- 类设计以 `08_类图设计.md` 为最高约束
- 服务骨架类设计以 `09_服务骨架类图设计.md` 为最高约束
- 产品化计划以 `10_插件产品化计划.md` 为阶段性约束
- 下一阶段任务以 `11_下一阶段任务清单.md` 为当前执行清单
- 数字人联动以 `12_数字人联动设计.md` 为扩展方向约束
- 明日人工测试以 `13_明日测试步骤.md` 为操作说明
- OmniVoice 参数体验以 `14_OmniVoice参数体验.md` 为参数说明
- 交付部署以 `15_交付与新机器部署.md` 为新机器部署说明
- 轻量 Demo UI 以 `16_轻量DemoUI设计.md` 为蓝图接线说明

如果后续要新增需求或改方案，必须先改文档，再改代码。
